# Good Pool Concept — Site vitrine

Site vitrine pour **Good Pool Concept**, spécialiste rénovation piscine (liner & PVC armé).

## Fonctionnalités

- Design moderne et épuré, palette or & marine cohérente avec le logo
- Module de chiffrage automatique de devis en 5 étapes
- Grille tarifaire PVC armé (rectangulaire, ovale, ronde, forme libre)
- Section coloris avec 6 ambiances
- Galerie de réalisations
- Section équipe & matériel
- Dépose de photos dans le formulaire de devis
- 100 % responsive (mobile, tablette, desktop)

## Structure

```
site giovanni/
├── index.html          # Structure du site
├── styles.css          # Design & responsive
├── calculator.js       # Moteur de devis et interactions
├── logo/               # Logo Good Pool Concept
├── images/             # Photos des piscines & matériel
└── banque d'image/     # Sources originales
```

## Lancer le site

Ouvrir `index.html` dans un navigateur — aucun serveur nécessaire, tout est statique.
